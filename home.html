What is the Document Object Model (DOM)?

The Document Object Model is a programming interface that allows us to create, change, or remove elements from the document.

How do you select an element from the DOM?

To select an element from the DOM, We can use methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), or querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a JavaScript technique where a single event listener is attached to a parent element to handle events for multiple child elements, including those added dynamically.Event delegation reduces the amount of code needed, making it easier to maintain and manage event handling logic, particularly when dealing with dynamically changing content. Attaching a single event listener to a parent is more efficient than attaching multiple listeners to individual child elements.

How do you manipulate an element's attributes and styles using the DOM?

I can change the value or content of an element by setting the innerText property of that element and add a new class attribute to an Element by using the add() method of the classList object or remove a class using the remove() method.
